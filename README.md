# IMDB_Review
MDB Movie Review Sentiment Analysis using NLP and Machine Learning. Built with Python, NLTK, TF-IDF, and Logistic Regression to classify reviews as positive or negative. Includes data preprocessing, visualization, model training, evaluation, and a Streamlit web app for real-time sentiment prediction.

Features
Data Cleaning and Preprocessing
Text Tokenization
Stopword Removal
Stemming/Lemmatization
TF-IDF Feature Extraction
Logistic Regression Model
Sentiment Prediction (Positive/Negative)
Interactive Streamlit Web Application

Tech Stack
Python
Pandas
NumPy
NLTK
Scikit-learn
Streamlit

Workflow
Load IMDB Dataset
Clean and preprocess text data
Perform Exploratory Data Analysis (EDA)
Convert text into numerical features using TF-IDF
Split data into training and testing sets
Train Logistic Regression model
Evaluate model performance
Save model and vectorizer
Deploy using Streamlit

Model Performance
Algorithm: Logistic Regression
Feature Extraction: TF-IDF Vectorizer
Task: Binary Sentiment Classification
Output: Positive or Negative Review
